Options

NOTE: Don't forget to click the "Save Changes" button below.

Ticker (Button Badge)


Reminder Notifications

Transient notifications that indicate the amount of time spent on the web so far today.


Reminder Notification Rate (In Minutes)

The number of minutes logged between each reminder notification.

Exclude These Sites (White List)

Enter a list of domains separated by commas to prevent those sites from being logged, and any time spent on them from being counted. For example: "wordpress.org, addons.mozilla.org, www.ubuntu.com" There is no need to include the "http://" or "https://" from the URL, but you must include any subdomains (such as "www.") when they are part of the site's address.

When a New Day Begins

Midnight is where the day begins, but you can adjust that by some number of hours. Enter "0" for midnight, "2" for 2:00 AM, "-1" for 11:00 PM, etc. (For the night owls.)
